Third-party asset starter set for future Crawl4AI branding replacement work.

Selected sources:
1. IconPark by ByteDance — Apache-2.0
   https://github.com/bytedance/IconPark
2. Fluent UI Emoji by Microsoft — MIT
   https://github.com/microsoft/fluentui-emoji

Notes:
- No CC BY assets are included in this starter set.
- Preserve upstream MIT/Apache license notices when redistributing downstream builds.
- IconPark assets are compact UI icons with simple blue/white fills and black strokes.
- Fluent UI Emoji assets are better for accent, splash, empty-state, or button-like decorative surfaces.
